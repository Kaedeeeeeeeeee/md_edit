传统 Markdown 编辑器要你看两份东西——左边写源码、右边看预览。Notation 不是这样：你直接编辑右边那种排好版的样子。磁盘上的文件依然是干净的 .md，能被任何工具打开，但你在 Notation 里看不到它的源码视图——也不需要切换。

文件夹即工作区

打开一个文件夹就是一个项目。子目录就是分组，文件就是笔记。系统沙盒下安全访问，最近列表自动记忆。

原生 macOS

Swift + SwiftUI 编写，约 5 MB 体积。macOS 26 上启用 Liquid Glass，macOS 15+ 保持原生菜单栏、系统深色模式和稳定材质 fallback。

键盘党的天堂

所有操作都有快捷键。⌘K 命令面板，⌘P 跳转文件。手不离键盘，思路不中断。
